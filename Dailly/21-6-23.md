# 表格相关

之前做过的心心念念的表格系统，今天偶然看到 [ag-grid](https://github.com.cnpmjs.org/ag-grid/ag-grid.git) 这个逆天的库，太强大了。

支持所有能想到的表格功能，并且性能优化到了极致，支持海量数据。

同时支持原生 js + vue + vue3 + react + angular 所有主流框架。

核心源码并不开源，想办法扒扒研究看看
